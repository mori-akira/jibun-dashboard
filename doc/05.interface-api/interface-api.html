userId
string

ユーザID

userName
required
string [ 1 .. 64 ] characters

ユーザ名

emailAddress
required
string <email> [ 1 .. 256 ] characters

eメールアドレス

Responses

http://localhost:4011/user

Content type
application/json

  • "userName": "string",
  • "emailAddress": "user@example.com"
}

application/json

      • "errorCode": "string",
      • "errorLevel": "DEBUG",
